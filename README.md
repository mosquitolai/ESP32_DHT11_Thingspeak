中文簡介：

本專題使用 ESP32 微控制器搭配 DHT11 溫濕度感測器，建立即時環境監測系統。系統可透過 WiFi 連線至網路，自動讀取周圍環境的溫度與濕度資料，並將數據上傳至 ThingSpeak 雲端平台進行儲存與視覺化分析。使用者可透過網頁隨時查看環境變化趨勢，達到遠端監控與資料紀錄的目的。此系統具有成本低、架構簡單、操作方便等優點，適合作為物聯網（IoT）應用的入門實作。

English Introduction:

This project uses an ESP32 microcontroller with a DHT11 temperature and humidity sensor to build a real-time environmental monitoring system. The system connects to the Internet via WiFi, automatically reads ambient temperature and humidity data, and uploads the information to the ThingSpeak cloud platform for storage and visualization. Users can view environmental trends anytime through a web interface, enabling remote monitoring and data recording. With its low cost, simple structure, and easy operation, this project is suitable as an introductory Internet of Things (IoT) application.
